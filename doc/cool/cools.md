### 2）💻 典型核心代码展示

#### 1. 自研 RAG 引擎核心 (RAGEngine.ts)
展示了基于关键词加权的轻量级知识检索算法，支持多模块（学习/生活/工作）索引。
```typescript
export class RAGEngine {
  public search(query: string, topK: number = 5): string {
    const queryLower = query.toLowerCase()
    const scored = this.items.map(item => {
      let score = 0
      const titleLower = item.title.toLowerCase()
      const contentLower = item.content.toLowerCase()
      // 权重：标题(10) > 内容(5) > 标签(3)
      if (titleLower.includes(queryLower)) score += 10
      if (contentLower.includes(queryLower)) score += 5
      item.tags.forEach(tag => {
        if (tag.toLowerCase().includes(queryLower)) score += 3
      })
      return { item, score }
    })
    return scored.filter(s => s.score > 0)
      .sort((a, b) => b.score - a.score).slice(0, topK)
      .map(r => `[${r.item.module}] ${r.item.title}\n${r.item.content}`).join('\n\n')
  }
}
```

#### 2.    (Character3D.tsx)
展示了通过 React 状态实时驱动 3D 角色的眨眼动画与基于语音播放状态的唇形同步。

```typescript
export function Character3D({ isSpeaking }) {
  // 随机眨眼动画逻辑
  useEffect(() => {
    const blinkInterval = setInterval(() => {
      setEyeBlinkState(0) // 闭眼
      setTimeout(() => setEyeBlinkState(1), 150) // 睁眼
    }, 3000 + Math.random() * 2000)
    return () => clearInterval(blinkInterval)
  }, [])

  // 模拟唇形同步 (Lip Sync)
  useEffect(() => {
    if (isSpeaking) {
      const lipSyncInterval = setInterval(() => {
        setMouthOpenness(Math.random() * 0.8 + 0.2)
      }, 100)
      return () => clearInterval(lipSyncInterval)
    }
    setMouthOpenness(0)
  }, [isSpeaking])
}
```

#### 3. 流式 AI 对话服务 (aiService.ts)
展示了对接 MiniMax-M2.5 API 的流式响应解析，支持 SSE 数据流的实时解析与回调。
```typescript
async streamChat(messages: ChatMessage[], onChunk: (chunk: string) => void) {
  const response = await fetch('https://api-inference.modelscope.cn/v1/chat/completions', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${apiKey}` },
    body: JSON.stringify({ model: 'MiniMax/MiniMax-M2.5', messages, stream: true }),
  })
  const reader = response.body?.getReader()
  while (true) {
    const { done, value } = await reader.read()
    if (done) break
    const chunk = new TextDecoder().decode(value)
    const lines = chunk.split('\n')
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const content = JSON.parse(line.slice(6)).choices[0]?.delta?.content
        if (content) onChunk(content) // 实时更新 UI 打字机效果
      }
    }
  }
}
```

#### 4. 动态主题切换引擎 (ThemeContext.tsx)
展示了 CSS 变量批量注入技术，实现 5 套风格主题的零闪烁瞬间切换。
```typescript
export function ThemeProvider({ children }) {
  useEffect(() => {
    const theme = themes[currentTheme]
    const root = document.documentElement
    // 动态注入全量 CSS 变量
    Object.entries(theme.colors).forEach(([key, value]) => {
      root.style.setProperty(`--color-${key}`, value)
    })
    // 快捷键支持：Shift + Q 全局循环切换主题
    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'q' && e.shiftKey) cycleTheme()
    }
    window.addEventListener('keydown', handleKeyDown)
    return () => window.removeEventListener('keydown', handleKeyDown)
  }, [currentTheme])
}
```

#### 5. 数字名片生成工具 (cardUtils.ts)
展示了基于 html2canvas 的 2x 高清截图技术，解决 Tailwind 渐变兼容性并生成图片。
```typescript
export async function generateCardImage(element: HTMLElement) {
  // 克隆 DOM 并预处理样式，确保截图效果一致
  const clone = element.cloneNode(true) as HTMLElement
  const canvas = await html2canvas(clone, {
    scale: 2,           // 高清采样
    useCORS: true,      // 处理跨域资源
    backgroundColor: null,
  })
  return canvas.toDataURL('image/png', 1.0)
}
```
